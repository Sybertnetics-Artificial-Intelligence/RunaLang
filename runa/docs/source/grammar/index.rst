EBNF Grammar
-----------

The Runa language is defined using Extended Backus-Naur Form (EBNF). The complete grammar specification can be found in the :doc:`ebnf` page.

For convenience, the grammar is divided into the following sections:

- Program Structure
- Comments
- Declarations and Assignments
- Expressions
- Collections
- Access Operations
- Control Structures
- Functions
- I/O Operations
- Modules
- Identifiers
- AI Extensions
- Knowledge Integration

Notation
-------

The grammar uses standard EBNF notation. See the :doc:`notation` page for details on the notation used.
