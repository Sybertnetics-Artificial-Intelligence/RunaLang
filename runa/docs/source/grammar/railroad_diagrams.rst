Railroad Diagrams
================

Railroad diagrams provide a visual representation of the Runa grammar. These diagrams make it easier to understand the syntax of the language.

.. note::
   Railroad diagrams will be generated automatically from the EBNF grammar. This page will be updated with the diagrams once they are generated.

Example Diagram
-------------

Below is an example of what the railroad diagrams will look like:

.. code-block:: text

   Statement
   ┌───────────────────┐
   │                   │
   ├─── Declaration ───┤
   │                   │
   ├─── Assignment ────┤
   │                   │
   ├─── Conditional ───┤
   │                   │
   ├─── Loop ──────────┤
   │                   │
   └───────────────────┘

The complete set of railroad diagrams will cover all aspects of the Runa grammar, including:

- Program structure
- Statements
- Expressions
- Control flow
- Functions
- Collections
- And more
