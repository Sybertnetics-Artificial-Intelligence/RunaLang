# HermodIDE

AI-powered IDE - Hermod's body. The interface through which Hermod manifests and interacts with users.

## Development Status

Currently in development. This is Hermod's IDE interface component.

## Architecture

- Electron-based desktop application
- Monaco Editor for code editing
- AI integration for intelligent assistance
- Runa language support

## Setup

```bash
npm install
npm run dev
```
