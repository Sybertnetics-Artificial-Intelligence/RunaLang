# Sybertnetics AI Solutions Monorepo

This monorepo contains the complete Sybertnetics AI Solutions ecosystem, including Runa programming language and HermodIDE.

## Project Structure

- **runa/** - Universal programming language with natural language syntax
- **hermod/** - AI-powered IDE (Hermod's body)
- **shared/** - Shared utilities and components
- **tools/** - Development and build tools
- **config/** - Configuration files
- **docs/** - Documentation
- **scripts/** - Utility scripts

## Quick Start

### Runa Language

cd runa
python runa_cli.py examples/hello_world.runa
