## Key Features

- **Natural Language Syntax**: Write code that reads like English
- **Performance Optimization**: Multiple optimization levels for efficient code execution
- **Loop Optimization**: Automatic optimization of loops for better performance
- **Multiple Target Languages**: Compile to JavaScript, Python, and other languages
